# SoluCX - Tech Test

This project is a Tech Test for the company SoluCX. \
It's developed in Lumen 5.8

## Configuration

Configure the `.env` for the application
```bash
  cd solucx-test
  cp .env.example .env
```

Run docker
```bash
  docker-compose up --build -d
```
